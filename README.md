# react-boilerplate
提供練習React+ Redux

## 安裝
  ```
    $git clone https://github.com/tingwei628/react-boilerplate.git // 找一個檔案夾, 把檔案抓下來
    
    $npm install    // 裝好套件
    
  ```
## 執行
  ```
    $npm start // 啟動 localhost:3000
    
  ```
## 套件內容
> React.js, Redux, React-router, Immutable.js
> Webpack,
> Eslint,
